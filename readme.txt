A simple WordPress plugin to integrate push notifications into your site.

== Description ==

This plugin allows you to send push notifications to your users' browsers. It includes basic setup for service workers and notification handling.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/push-notification-integration` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The plugin will automatically enqueue necessary scripts and register the service worker.

== Frequently Asked Questions ==

= How do I customize the notifications? =

Edit the JavaScript files in the plugin directory to suit your needs.
